# ЛабКонтроль

Демонстрационный фронтенд системы проверки лабораторных работ на C++ на заимствования.

## Запуск

Откройте `index.html` в любом современном браузере. Установка зависимостей не требуется.

Для запуска через локальный сервер можно использовать:

```powershell
npx serve .
```

## Возможности

- загрузка исходников CPP, CC, H, HPP и ZIP-архивов проектов;
- демонстрация анализа токенов, AST и нормализации идентификаторов;
- демонстрация процесса проверки;
- отчёт с процентом оригинальности и источниками совпадений;
- история и фильтрация проверок;
- экраны дисциплин, аналитики и базы знаний;
- адаптивная вёрстка для компьютеров и смартфонов.
